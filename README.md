## ✨ Características

- 📊 **Dashboard financiero** con KPIs de balance, ingresos, gastos y activos
- 💸 **Transacciones** con tipo (ingreso/gasto), categorías, cuenta y fecha
- 🧾 **Facturación** con estado (pagada / pendiente / vencida) y resumen por colores
- 🏦 **Cuentas** bancarias, efectivo y por cobrar con barra de distribución
- 📈 **Reportes** con margen bruto, ratio de liquidez, tendencias y categorías
- 🔍 **Búsqueda y filtros** por tipo (ingreso/gasto) y descripción/categoría
- 🌗 **Modo claro/oscuro** con balance permanente visible en la sidebar
- 📥 **Exportación CSV** de transacciones y facturas
- ✅ **Marcar factura como pagada** en un solo click

## 🛠️ Stack tecnológico

- **React 18** — hooks, `useMemo`, `useCallback`
- **Vite 5** — bundler con HMR
- **Recharts** — Bar, Area, Pie
- **CSS-in-JS** — estilos inline temáticos
- **GitHub Actions + GitHub Pages** — CI/CD

## 💻 Instalación local

```bash
git clone https://github.com/Juanmaya25/contafacil-accounting.git
cd contafacil-accounting
npm install
npm run dev
```

Abre [http://localhost:5173/contafacil-accounting/](http://localhost:5173/contafacil-accounting/).
